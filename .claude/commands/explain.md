# Explain Formula

Trace and explain how a cell's value is calculated.

## Instructions

1. Read the formula in the specified cell
2. Identify all references (same-sheet and cross-sheet)
3. Recursively trace dependencies to source data
4. Build a dependency tree showing:
   - Direct inputs
   - Intermediate calculations
   - Source sheets/cells
5. Explain the business logic in plain English

## Arguments

$ARGUMENTS - Cell reference like "Monthly Summary!C12" or "'Costs by Department'!C46"
