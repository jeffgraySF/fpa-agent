# Modify Sheet

Make structural or formula changes to the spreadsheet.

## Instructions

1. Read INSTRUCTIONS.md for formula standards
2. Before making changes:
   - Read the affected area to understand current structure
   - Plan the changes (new rows, formula updates, etc.)
   - Identify cross-sheet references that may need updating
3. Make the changes using SheetsClient
4. After changes:
   - Verify no #REF! or other errors introduced
   - Update template_specs.md if structure changed
   - Update INSTRUCTIONS.md if new formula patterns added
5. Summarize what was changed

## Arguments

$ARGUMENTS - Description of the modification to make

Always confirm the plan before executing destructive changes.
